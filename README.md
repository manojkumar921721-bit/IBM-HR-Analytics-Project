# IBM HR Attrition Analysis

## Project Overview

This project analyzes employee attrition patterns using the IBM HR Analytics dataset.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* SQL
* Power BI

## Key Insights

* Sales Representatives showed the highest attrition rate.
* Research & Development had the highest number of employees leaving.
* Overtime employees were more likely to leave the organization.
